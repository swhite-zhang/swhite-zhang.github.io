处理错误

语法错误（Syntax Error）

违反了编程语言的语句形式，或使用规则，一般由编译器检查指出。

常见的语法错误

  • 写错关键字
  • 遗漏分号
  • 遗漏括号
  • 遗漏引导

运行时错误（Runtime Error）

程序运行时，遇到一个无法执行的操作。编译器无法检查指出。

逻辑错误（Logic Error）

程序运行的结果和预期不符。编译器无法检查指出。

输入和输出（Input and Output）

C++使用流的概念对屏幕、键盘或文件进行输入输出操作。

标准I/O（standard I/O）

流 描述 操作符 函数
cin 标准输入流 >> cin.get()从流中读取一个字符,cin.getline()从六种读取字符直至换行符或指定分隔符,cin.ignore()从六种读取并舍弃指定数量的字符
cout 标准输出流 << cout.put()将字符写到流中,cout.flush()将流中缓存内容全部输出
cerr 标准错误流
clog 标准错误流
